
------------------- SOBRE -------------------

Um sistema interativo em terminal (CLI) desenvolvido em Python para gerenciar e anotar as compras de café da equipe, garantindo o controle de quem trouxe o pão ou a nata da vez.

O Café da Firma foi criado para automatizar e organizar o registro de compras de itens de café da manhã de uma equipe. O programa conta com uma interface personalizada diretamente no terminal do Windows, utilizando cores e artes em formato ASCII para tornar a experiência mais amigável e divertida.

------------------- FUNCIONALIDADES -------------------

[x] Interface customizada com arte ASCII e alteração de cores no terminal.

[x] Sistema de autenticação de usuário e senha.

[x] Tratamento de erros robusto para entradas inválidas (letras onde deveriam ser números, etc.).

[x] Menu principal de navegação pós-login.

[x] Registro automatizado de compras de produtos específicos (Pão, Nata ou Ambos).

[ ] Visualização da relação de compras consolidada da equipe (em desenvolvimento).

[ ] Banco de Dados para armazenamento dos dados (em desenvolvimento).

------------------- TECNOLOGIAS UTILIZADAS -------------------

LINGUAGEM PRINCIPAL: Python 3

MODULOS NATIVOS: time, os, datetime

MODULOS INTERNOS: Recursos customizados de interface e banco de dados (recursos.funcoes)
